# A collection of HPC dwarf reference implementations

This is a collection of portable C benchmark implementations for the HPC dwarfs algorithmic classes as initially proposed by the report [The Landscape of Parallel Computing Research: A View from Berkeley](https://www2.eecs.berkeley.edu/Pubs/TechRpts/2006/EECS-2006-183.pdf). Each benchmark is designed for performance engineering: compact, self-contained, and focused on a single computational pattern.

| Dwarf | Benchmark |
|---|---|
| Sparse Linear Algebra | [SparseBench](https://github.com/HPC-Dwarfs/SparseBench) |
| N-Body Methods | [MD-Bench](https://github.com/HPC-Dwarfs/MD-Bench) |
| Structured Grids + Sparse Linear Algebra | [CFD-Bench](https://github.com/HPC-Dwarfs/CFD-Bench) |
| Structured Grids | [LBM-Bench](https://github.com/HPC-Dwarfs/LBM-Bench) |
| Spectral Methods | DFT-Bench *(coming soon)* |
| Signal Processing | [RabbitCT](https://github.com/HPC-Dwarfs/RabbitCT) |
| Streaming Kernels | [TheBandwidthBenchmark](https://github.com/HPC-Dwarfs/TheBandwidthBenchmark) |

This is an effort by [NHR@FAU](https://nhr.fau.de)
